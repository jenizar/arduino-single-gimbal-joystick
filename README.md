# arduino-single-gimbal-joystick
DIY Arduino Single Gimbal Joystick Control Car At Home

materials:

2x arduino nano

1x l298n motor driver

2x nrf24l01 pa radio module

1x joystick module

4x motor dc bo

4x wheel rc

1x lipo battery 2s 7.4v 25c

2x pcb

1x cardboard
